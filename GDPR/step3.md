# Install 'CookieYes' Plugin

Please follow the below instructions to install 'CookieYes' Plugin.

## 1. Go to plugin page and click the 'Add new' button.
![Alt text](https://raw.githubusercontent.com/KuroP1/katacoda-scenarios/blob/main/GDPR/images/step3-1.PNG "a title")

## 2. Enter 'CookieYes' in the search bar.
- ![Alt text](https://raw.githubusercontent.com/KuroP1/katacoda-scenarios/blob/main/GDPR/images/step3-2.PNG "a title")

## 3. Click the 'Install' button to install the 'CookieYes' plugin.
- ![Alt text](https://raw.githubusercontent.com/KuroP1/katacoda-scenarios/blob/main/GDPR/images/step3-3.PNG "a title")

## 4. Click the `Activate` button to activate the plugin.
- ![Alt text](https://raw.githubusercontent.com/KuroP1/katacoda-scenarios/blob/main/GDPR/images/step3-4.PNG "a title")

## 5. Now you can see the shortcut of the 'CookieYes' plugin on the left-hand side.
- ![Alt text](https://raw.githubusercontent.com/KuroP1/katacoda-scenarios/blob/main/GDPR/images/step3-5.PNG "a title")
